¿Cómo funciona la virtualización?




La virtualización ejecuta los procesos en una máquina física de otras máquinas virtuales; es decir; consigue que una computadora trabaje como muchas computadoras juntas; ¿de qué manera?: el software o hipervisor separa el sistema operativo de las aplicaciones del hardware. Esto trae como resultado que cada máquina virtual ejecute su propio sistema operativo y se comporte como una computadora independiente, a pesar de que se encuentra instalado en un solo hardware.

Existen diferentes tipos de virtualización:

Virtualización de datos. La virtualización de datos combina datos de fuentes (base de datos y almacenamiento, computación en la nube, software como servicio, big data, xml, excel, pdf, docs, emails, colaboración web 2.0) en formatos digeribles (Aplicaciones empresariales, ESB, reportes, BI o inteligencia de negocios), acelerando la eficiencia de la minería de datos Permite ahorrar costos de soporte y disminuye el número de emisores fijos.

Virtualización de escritorio. El hardware y software de la CP se separan. Funciona para mantener los escritorios individuales en un único servidor central.

Virtualización del servidor. Con la ayuda de un software especializado, un administrador puede convertir un servidor físico en múltiples máquinas virtuales, por lo que cada servidor actúa como dispositivo físico único.

Virtualización del sistema operativo. Se trata de implementar un sistema operativo capaz de ejecutar diferentes aplicaciones manejadas por múltiples usuarios en una sola computadora a la vez.

Virtualización de funciones de red. Aquí las funciones de enrutamiento, equilibrio de carga y cortafuegos se empaquetan como máquinas virtuales dentro de un hardware.
